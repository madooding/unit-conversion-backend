## Installation

```bash
$ npm install
```

## Environment Variables

Create a new file ```.env``` in root directory then copy below content

```
NODE_ENV=development
NODE_PORT=3000
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
